How to Chabge Default Password

1. At your first login, type in your registered email and the default password provided in the confirmation email.

2. After logging in successfully, the Change Password modal will appear automatically. Fill up the entry boxes that constitute the parameters in changing the default password (See Figure 1-1).


Figure 1-1 Change Password Scheme Form


3. Thyp in the Current Password, New Password, and Confirm Password on the entry boxes provided.

4. When done, click the Update Password button. You will now be directed to the Login page of the system.
