How to Edit IS Strategy

1. To Edit IS Strategy, click the IS Strategy button as shown in Figure 5-10. Fill-up the entry boxes that constitute the parameters for updating the existing IS Strategy (See Figure 5-10).


Figure 5-10 IS Strategy Scheme Form


2. To upload the IS Interface, click the Browse button. A File Upload window will appear. Locate the image file to be uploaded, then click the Open button.

3. Click the Add New button to add Proposed Information System. Fill-up the entry boxes that constitute the parameters for adding Proposed Information System (See Figure 5-11).

4. Click the Add New button to add Database Required. Fill-up the entry boxes that constitute the parameters for adding Database Required (See Figure ______________).

5. To upload the Network Layout, click the Browse button. A File Upload window will appear. Locate the image file to be uploaded, then click the Open button.

6. When done, click the Save button to update the existing IS Strategy.
