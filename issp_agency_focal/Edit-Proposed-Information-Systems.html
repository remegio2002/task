How to Edit Proposed Information Systems

1. To edit Proposed Information Systems, click the Table Row on the Proposed Information Systems (See Figure 5-10). Fill-up the entry boxes that constitute the parameters for updating the existing Proposed Information Systems (See Figure 5-12).


Figure 5-12 Proposed Information Systems Scheme Form


2. Select the Parent Information System on the dropdown list.

3. Type in the Name of Information System / Sub-System on the entry box provided.

4. Select the IS Classification and Database Required on the dropdown lists.

5. Click the Plus (+) icon button to add Database Required. Fill-up the entry boxes that constitute the parameters for adding Database Required (See Figure 5-15).

6. Type in the Description on the entry box provided.

7. Select the From and To for the Implementation Schedule on the datepicker.

8. Select the Status and Development Strategy on the dropdown lists.

9. Type in the Computing Scheme on the entry box provided.

10. Type in the Internal Users, External Users, and System Owner on the entry boxes provided.

11. When done, click the Save button to update the existing Proposed Information System or the Close (x) icon button to close the form.
