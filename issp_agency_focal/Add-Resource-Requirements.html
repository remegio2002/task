How to Add Resource Requirements

1. To add Resource Requirements, click the Add New button (See Figure 5-24). Fill-up the entry boxes that constitute the parameters for adding new Resource Requirement (See Figure 5-25).


Figure 5-25 Resource Requirements Scheme Form


2. Select the ICT Item and Office on the dropdown lists.

3. Select the Year on the datepicker.

4. Type in the Number of Units on the entry box provided.

4. Click the Plus (+) icon button to add new entry boxes for the Year and Number of Units.

6. When done, click the Save button to add new Resource Requirement or the Cancel button to close the form.
