How to Add Database Required

1. To add Database Required, click the Add New button (See Figure 5-10). Fill-up the entry boxes that constitute the parameters for adding new Database Required (See Figure 5-15).


Figure 5-15 Database Required Scheme Form


2. Type in the Name of Database, Description, and Information System/s Served on the entry boxes provided.

3. Select the From and To for the Implementation Schedule on the datepicker.

4. Select the Status on the dropdown list.

5. Type in the Data Archiving / Storage Media, Owner, Internal Users, and External Users on the entry boxes provided.

6. When done, click the Save button to add new Database Required or the Cancel button to close the form.
