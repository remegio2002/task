How to Delete Database Required

1. To delete Database Required, click the Table Row on the Database Required (See Figure 5-10). Click the Delete button (See Figure 5-17).


Figure 5-17 Delete Database Required


2. Once you click the Delete button, delete dialog box will pop-up (See Figure 5-18).


Figure 5-18 Delete Dialog Box


3. Click the Delete button to proceed or click the Cancel button to do otherwise. A confirmation message will be displayed if the action is successful.
