How to Edit Database Required

1. To edit Database Required, click the Table Row on the Database Required (See Figure 5-10). Fill-up the entry boxes that constitute the parameters for updating the existing Database Required (See Figure 5-16).


Figure 5-16 Database Required Scheme Form


2. Type in the Name of Database, Description, and Information System/s Served on the entry boxes provided.

3. Select the From and To for the Implementation Schedule on the datepicker.

4. Select the Status on the dropdown list.

5. Type in the Data Archiving / Storage Media, Owner, Internal Users, and External Users on the entry boxes provided.

6. When done, click the Save button to update the existing Database Required or the Close (x) icon button to close the form.
