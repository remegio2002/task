How to Add Performance Measurement Framework

1. To add Performance Measurement Framework, click the Add New button (See Figure 5-30). Fill-up the entry boxes that constitute the parameters for adding new Performance Measurement Framework (See Figure 5-31).


Figure 5-31 Performance Measurement Framework Scheme Form


2. Select the Hierarchy Type on the dropdown list.

3. Type in the Hierarchy of Targeted Results, Data Collection Methods, Responsible to Collect Data, OVI, Baseline Data, and Targets on the entry boxes provided.

4. Click the Plus (+) icon button to add new entry boxes for OVI, Baseline Data and Targets.

5. When done, click the Save button to add new Performance Measurement Framework or the Cancel button to close the form.
