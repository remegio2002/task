How to Delete Performance Measurement Framework

1. To delete Performance Measurement Framework, click the Table Row as shown in Figure 5-30. Click the Delete button (See Figure 5-33).


Figure 5-33 Delete Performance Measurement Framework


2. Once you click the Delete button, delete dialog box will pop-up (See Figure 5-34).


Figure 5-34 Delete Dialog Box


3. Click the Delete button to proceed or click the Cancel button to do otherwise. A confirmation message will be displayed if the action is successful.
