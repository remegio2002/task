How to Edit Grants-In-Aid (GIA) Projects

1. To edit Grants-In-Aid (GIA) Projects, click the Table Row as shown in Figure 5-35. Fill-up the entry boxes that constitute the parameters for updating the existing Grants-In-Aid (GIA) Projects (See Figure 5-37).


Figure 5-37 Grants-In-Aid (GIA) Projects Scheme Form


2. Type in the Name / Title of ICT Project on the entry box provided.

3. Select the From and To for the Duration on the datepicker.

4. Type in the Objectives and Deliverables on the entry boxes provided.

5. When done, click the Save button to update the existing Grants-In-Aid (GIA) Projects or the Close (x) icon button to close the form.
