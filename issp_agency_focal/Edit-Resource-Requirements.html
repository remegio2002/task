How to Edit Resource Requirements

1. To edit Resource Requirements, click the Table Row on the ICT Items as shown in Figure 5-25. Fill-up the entry boxes that constitute the parameters for updating the existing Resource Requirement (See Figure 5-27).


Figure 5-27 Resource Requirements Scheme Form


2. Select the ICT Item and Office on the dropdown lists.

3. Select the Year on the datepicker.

4. Type in the Number of Units on the entry box provided.

4. Click the Plus (+) icon button to add new entry boxes for Year and Number of Units.

6. When done, click the Update button to update the existing Resource Requirement or the Close (x) icon button to close the form.
