How to Add Continuing Costs

1. To add Continuing Costs, click the Add New button (See Figure 5-56). Fill-up the entry boxes that constitute the parameters for adding new Continuing Cost (See Figure 5-57).


Figure 5-57 Continuing Costs Scheme Form


2. Type in the Name of IS/Database/ICT Project on the entry box provided.

3. Select the ICT Item, Allotment Class, and Office on the dropdown lists.

4. Select the Year on the datepicker.

5. Type in the Number of Units on the entry box provided.

6. Click the Plus (+) icon button to add new entry boxes for Year and Number of Units.

7. When done, click the Save button to add new Continuing Cost or the Cancel button to close the form.
