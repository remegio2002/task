How to Delete User Account

1. To delete User Account, click the Table Row as shown in Figure 7-31. Click the Delete button as shown in Figure 7-35.


Figure 7-35 Delete User Account Scheme Form


2. Once you click the Delete button, delete dialog box will pop-up (See Figure 7-36).


Figure 7-36 Delete Dialog Box


3. Click the Delete button to proceed or click the Close (x) icon button to do otherwise. A confirmation message will be displayed if the action is successful.
