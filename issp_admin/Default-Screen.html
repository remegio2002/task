Default Screen

Once logged in to the system you will be automatically directed to the Dashboard Section (See Figure 2).


Figure 2 Default Page


These are the components of the Default Screen and a brief description of what they are:


Displays the charts of Budget, Human Resources, Information Systems, ICT Projects, and the list of ISSP Submission Status.
Provides links to the Agency Profile and Mandate.
Provides links to the Organizational Structure, IS Strategy, ICT Projects, Resource Requirements and Infrastructure Inventory.
Provides links to preview Information Systems Strategic Plan Report.
Provides links to the Agency/Institution, Funding Source, ICT Categories, ICT Items and User Accounts.
Displays the name of the logged-in user, provides options to view the user's profile and Logout sub-section.
Provides links to Online Help, FAQS and About sub-sections.
