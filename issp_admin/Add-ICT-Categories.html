How to Add ICT Categories

1. To add ICT Categories, click the Add New button as shown in Figure 7-13. Fill-up the entry box that constitute the parameter for adding new ICT Categories (See Figure 7-15).


Figure 7-15 Add ICT Categories Scheme Form


2. Type in the ICT Item Category on the entry box provided.

3. When done, click the Save button to add new ICT Categories or the Cancel button to cancel the operation.
