How to Delete ICT Categories

1. To delete ICT Categories, click the Table Row as shown in Figure 7-13. Click the Delete button as shown (See Figure 7-17).


Figure 7-17 Delete ICT Categories Scheme Form


2. Once you click the Delete button, delete dialog box will pop-up (See Figure 7-18).


Figure 7-18 Delete Dialog Box


3. Click the Delete button to proceed or click the Close (x) icon button to do otherwise. A confirmation message will be displayed if the action is successful.
