How to Add Agency/Institution

To add Agency/Institution, click the Add New button as shown in Figure 7-1. Fill-up the entry boxes that constitute the parameters for adding new Agency/Insitution (See Figure 7-3).


Figure 7-3 Add Agency/Institution Scheme Form


1. To upload the image file, click Browse button. A File Upload window will appear. Locate the image file to be uploaded, then click Open button.

2. Type in the Agency Name and Alias (short name) on the entry boxes provided.

3. Select the Agency Group on the dropdown list.

4. Type in the Agency Official Website Link and Name of Agency Head (First Name, M.I., Surname, Suffix) on the entry boxes provided.

5. When done, click the Save button to add new Agency/Institution or the Cancel button to cancel the operation.
