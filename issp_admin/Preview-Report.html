How to Preview Information Systems Strategic Plan

1. To preview the Report, click the Report button as shown in Figure 6. Fill-up the entry boxes that constitute the parameters for generating report in Figure 6-1.


Figure 6-1 Preview Report


2. Type in the Type of Report on the entry box provided.

3. Select the Calendar Year and ISSP Sectin on the dropdown list.

4. When done, click the Preview button to Generate Report.
