How to Edit ICT Categories

1. To edit ICT Categories, click the Table Row as shown in Figure 7-13. Fill-up the entry boxes that constitute the parameters for updating Funding Source (See Figure 7-16).


Figure 7-16 Edit ICT Categories Scheme Form


2. Type in the ICT Item Category on the entry box provided.

3. When done, click the Save button to update ICT Categories or the Close (x) icon button to cancel the operation.
