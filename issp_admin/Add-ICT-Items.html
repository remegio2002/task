How to Add ICT Items

1. To add ICT Items, click the Add New button as shown in Figure 7-19. Fill-up the entry boxes that constitute the parameters for adding new ICT Items (See Figure 7-21).


Figure 7-21 Add ICT Items Scheme Form


2. Select the Item Category and Parent Item on the dropdown lists.

3. Type in the Item Name, Item Specifications, and Estimated Cost on the entry boxes provided.

4. Type in the Unit of Measure on the entry box provided or select the Unit of Measure on the dropdown list.

5. When done, click the Save button to add new ICT Item or the Cancel button to cancel the operation.
