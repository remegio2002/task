How to Add IS Classification

1. To add IS Classification, click the Add New button as shown in Figure 7-25. Fill-up the entry box that constitute the parameter for adding new IS Classification (See Figure 7-27).


Figure 7-27 Add IS Classification Scheme Form


2. Type in the IS Classification on the entry box provided.

3. When done, click the Save button to add new IS Classification or the Cancel button to cancel the operation.
