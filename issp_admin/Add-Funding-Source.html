How to Add Funding Source

1. To add Funding Source, click the Add New button as shown in Figure 7-7. Fill-up the entry boxes that constitute the parameters for adding new Funding Source (See Figure 7-9).


Figure 7-9 Add Funding Source Scheme Form


2. Type in the Funding Source Code and Funding Source Name on the entry boxes provided.

3. When done, click the Save button to add new Funding Source or the Cancel button to cancel the operation.
