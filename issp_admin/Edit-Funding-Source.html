How to Edit Funding Source

1. To edit Funding Source, click the Table Row as shown in Figure 7-7. Fill-up the entry boxes that constitute the parameters for updating Funding Source (See Figure 7-10).


Figure 7-10 Edit Funding Source Scheme Form


2. Type in the Funding Source Code and Funding Source Name on the entry boxes provided.

3. When done, click the Save button to update Funding Source or the Close (x) icon button to cancel the operation.
