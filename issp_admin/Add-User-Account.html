How to Add User Account

1. To add User Account, click the Add New button as shown in Figure 7-31. Fill-up the entry boxes that constitute the parameters for adding new User Account (See Figure 7-33).


Figure 7-33 Add User Account Scheme Form


2. Type in the Employee Name (First Name, M.I., Surname, Suffix), Email Address, Contact Number, and Position on the entry boxes provided.

3. Select the Access Level and Agency on the dropdown lists.

4. When done, click the Invite button to add new User Account or the Cancel button to cancel the operation.


Note: An email notification will be sent to the registered email address, including the default password.
