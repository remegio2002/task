How to Initiate Password Reset

1. To initiate a password reset, click the Forgot Password? link on the Login page (See Figure 1). Fill-up the entry box that constitute the parameter for password reset (See Figure 1-1).


Figure 1-1 Forgot Password Scheme Form


2. Type in the Email Address on the entry box provided.

3. When done, click the Send button.


Note: If send email is successful, you will receive an email notification explaining how to reset your password.


4. Click the Reset Password link sent to your email. Fill-up the entry boxes provided that constitute the parameters for resetting your password (See Figure 1-2).


Figure 1-2 Reset Password Scheme Form


5. Type in the New Password and Confirm Password on the entry boxes provided.

6. When done, click the Reset Password button to reset the password.


Note: Upon successful password reset, a confirmation message will be displayed, and you will be directed to the Login page.
