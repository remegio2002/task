ISSP Module

The RMIS Module is one of DOST-CO RMIS component that allows ISO Auditor to facilitate the collection of risk data and provides the ISO Auditor to track and manage reports capabilities to monitor and control the overall cost of risk management.


The ISSP Module allows the user to:


View charts for Budget, Budget, Human Resources, Information Systems, and ICT Projects
View the list of ISSP Submission Status
Manage Agency Profile
Manage Agency Mandate
Manage Organizational Structure
Manage IS Strategy
Manage ICT Projects
Manage Resource Requirements
View Information System Strategic Plan Report
Manage Agency/Institution
Manage Funding Source
Manage ICT Categories
Manage ICT Items
Manage User Accounts
View User Profile
Logout from the system
View Online manual
View list of frequently Ask question
View system description and credits
