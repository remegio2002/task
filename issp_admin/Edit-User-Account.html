How to Edit User Account

1. To edit User Account, click the Table Row as shown in Figure 7-31. Fill-up the entry boxes that constitute the parameters for updating User Account (See Figure 7-34).


Figure 7-34 Edit User Account Scheme Form


2. Type in the Employee Name (First Name, M.I., Surname, Suffix), Email Address, Contact Number, and Position on the entry boxes provided.

3. Select the Access Level and Agency on the dropdown lists.

4. When done, click the Save button to update User Account or the Close (x) button to cancel the operation.
