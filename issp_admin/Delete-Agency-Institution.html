How to Delete Agency/Institution

1. To delete Agency/Institution, click the Table Row as shown in Figure 7-1. Click the Delete button as shown (See Figure 7-5).


Figure 7-5 Edit Agency/Institution Scheme Form


2. Once you click the Delete button, delete dialog box will pop-up (See Figure 7-6).


Figure 7-6 Delete Dialog Box


3. Click the Delete button to proceed or click the Cancel button to do otherwise. A confirmation message will be displayed if the action is successful.
