How to Edit Agency/Institution

1. To edit Agency/Institution, click the Table Row as shown in Figure 7-1. Fill-up the entry boxes that constitute the parameters for updating Agency/Insitution (See Figure 7-4).


Figure 7-4 Edit Agency/Institution Scheme Form


2. To upload the Agency Logo, click the Browse button. A File Upload window will appear. Locate the image file to be uploaded, then click the Open button.

3. Type in the Agency Name and Alias (short name) on the entry boxes provided.

4. Select the Agency Group on the dropdown list.

5. Type in the Agency Official Website Link and Name of Agency Head (First Name, M.I., Surname, Suffix) on the entry boxes provided.

6. When done, click the Save button to update Agency/Institution or the Close (x) icon button to cancel the operation.
